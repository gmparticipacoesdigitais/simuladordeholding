Gerador de Hash SHA-256

Use esta ferramenta para gerar hashes SHA-256 de emails para adicionar à allowlist do simulador.

Como usar:

  1. 1. Digite o email que deseja autorizar
  2. 2. Clique em "Gerar Hash"
  3. 3. Copie o hash gerado
  4. 4. Adicione o hash à constante ALLOWLIST_SHA256 no arquivo index.html
